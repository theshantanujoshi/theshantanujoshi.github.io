# ✨ Shantanu Joshi | Minimal Portfolio

A high-end, minimal social links landing page designed with a premium aesthetic. Featuring a dynamic PixelBlast background, custom cursor trails, and a sleek glassmorphism UI.

![Portfolio Preview](https://images.unsplash.com/photo-1599148400620-8e1ff0bf28d8?q=80&w=2268&auto=format&fit=crop)

## 🚀 Features

- **PixelBlast Background**: A custom Three.js powered interactive background with liquid ripple effects.
- **Custom Cursor**: A sophisticated cursor with real-time pixel trails that respond to movement.
- **Glassmorphism UI**: Beautifully blurred cards and buttons with subtle borders and glow effects.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Micro-interactions**: Hover-triggered glows and smooth transitions for a premium feel.
- **Minimalist Aesthetics**: Clean typography using Inter and JetBrains Mono.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla CSS3
- **Animations**: [Three.js](https://threejs.org/), [Postprocessing](https://github.com/vanruesc/postprocessing)
- **Icons**: [Feather Icons](https://feathericons.com/)
- **Fonts**: [Google Fonts](https://fonts.google.com/) (Inter, JetBrains Mono)

## 💻 Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/servernayachast/portfolio.git
   ```
2. Open `index.html` in your browser.
3. (Optional) Use a local server like Live Server in VS Code for the best experience.

## 🎨 Customization

To personalize this portfolio:
- Update the profile information in the `<main>` section of `index.html`.
- Replace `profilepicture.jpg` with your own image.
- Modify the `links` div to include your own social profiles.
- Adjust the CSS variables in `:root` to change the color palette.
